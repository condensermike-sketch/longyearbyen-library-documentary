# Svalbard Library Documentary Reference Hub — Version 2

This version converts the original one-page site into a multi-page documentary production reference site.

## Pages

- `index.html` — Overview
- `research.html` — Research notes and watchlist
- `preproduction.html` — Permissions, interviews, story planning
- `production.html` — Shot planning and production notes
- `gear.html` — Equipment, budget, purchase priorities
- `howtos.html` — Practical filmmaking guides
- `links.html` — Helpful external links
- `checklists.html` — Travel and daily production checklists

## How to use

Open `index.html` in a web browser.  
Edit the HTML files directly to add your own notes, contacts, links, and production details.

The same CSS layout and colors from Version 1 have been retained.
